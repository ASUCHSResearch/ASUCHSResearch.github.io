# CHS Research Day Poster Companion

Static site for browsing CHS Research Day posters and opening poster-specific GPT prompts.

## Runtime Files

The deployed site only needs:

- `index.html`
- `posters/index.json`
- `posters/*.json`

## What The Site Does

- shows a searchable poster directory
- filters posters by curated category
- opens a detail page for each poster
- lets attendees copy a GPT seed prompt
- routes users to the A-M or N-Z Custom GPT
